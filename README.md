# Network Systems — CompTIA Network+ Learning Repository

A structured, self-paced curriculum built around **CompTIA Network+ Guide to Networks, 9th Edition** (Jill West) — organized by the book's 12 modules, with notes, exercises, interactive quizzes, and hands-on projects.

## Structure

| Folder | Contents |
|--------|----------|
| [01-notes](01-notes/README.md) | Topic-based markdown notes (several per module) |
| [02-exercises](02-exercises/README.md) | Practice exercises with worked solutions |
| [03-quiz](03-quiz/README.md) | Interactive MCQ quizzes — [open quiz site](https://ijk37.github.io/network-systems/03-quiz/) |
| [04-projects](04-projects/README.md) | Hands-on Packet Tracer labs + design/analysis projects |
| [05-resources](05-resources/) | Lecture notes, reference material, and lab files |

## Modules Covered

1. Introduction to Networking
2. Infrastructure & Documentation
3. Addressing
4. Network Protocols
5. Cabling
6. Wireless Networking
7. Cloud Computing & Remote Access
8. Subnets & VLANs
9. Network Risk Management
10. Security in Network Design
11. Network Performance & Recovery
12. Wide Area Networks

## 🧪 Quiz

**[👉 Open the Quiz Hub](https://ijk37.github.io/network-systems/03-quiz/)** — 12 chapter quizzes + 5 cumulative mixed quizzes, with instant scoring and full end-of-quiz review.

- **~1,200** chapter questions (~100 per module); each attempt draws **25 random**, with options shuffled so no two attempts are alike
- **5** mixed quizzes (50 / 50 / 75 / 100 / 100), re-mixed from the module pools
- Skip / back navigation, and every attempt re-draws a fresh set

## How to Use

1. **Read** a note in [`01-notes/`](01-notes/README.md)
2. **Practice** with the matching exercise in [`02-exercises/`](02-exercises/README.md)
3. **Test** yourself → [Quiz Hub](https://ijk37.github.io/network-systems/03-quiz/)
4. **Build** something real in [`04-projects/`](04-projects/README.md)

## Prerequisites

- **Cisco Packet Tracer** (free) for the lab projects
- **Wireshark** (free) for the traffic-analysis project
- Basic computer literacy

## Notes on Content

- Numbering follows the **book's module order** (Module 08 = Subnets & VLANs, etc.).
- Quiz sizes are weighted so core exam topics (Addressing, Protocols, Subnets/VLANs, Security) carry more questions than lighter ones.
- `05-resources/` originals (PDF/PPTX/DOCX) are git-ignored; their `.md` conversions are tracked.
